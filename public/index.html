📡 Backend API - Dominos Store 👨‍💻En Desarrollo👨‍💻

Bienvenido al backend del ecommerce "Dominos Store". Aquí encuentras una descripción rápida del estado del servidor y sus rutas disponibles.

📦 Descripción

API REST construida con Node.js, Express y Sequelize (ORM con migraciones), conectada a MySQL. Esta API permite manejar usuarios, roles, y próximamente productos, carritos y pedidos.

🚀 Rutas disponibles (versión v1)

🔐 Rutas de Roles

  • GET /api/v1/roles → Lista todos los roles
  • POST /api/v1/roles → Crea un nuevo rol
  • PUT /api/v1/roles/:id → Actualiza un rol por ID
  • DELETE /api/v1/roles/:id → Elimina un rol por ID

👤 Rutas de Usuarios

  • GET /api/v1/users → Lista todos los usuarios
  • GET /api/v1/users/:id → Muestra un usuario por ID
  • POST /api/v1/users → Crea un nuevo usuario
  • PUT /api/v1/users/:id → Actualiza un usuario
  • DELETE /api/v1/users/:id → Elimina un usuario

🛒 Rutas de Productos

  • GET /api/v1/products → Lista todos los productos
  • GET /api/v1/products/:id → Muestra un producto por ID
  • POST /api/v1/products → Crea un nuevo producto
  • PUT /api/v1/products/:id → Actualiza un producto por ID
  • DELETE /api/v1/products/:id → Elimina un producto por ID

🛒 Rutas del carrito

  • GET /api/v1/carts → Lista todos los carritos

🛠 Tecnologías utilizadas hasta el momento

  • Node.js
  • Express.js
  • MySQL
  • Sequelize CLI (con migraciones)
  • Bcrypt
  • Json Web Token (JWT)

📄 Estado actual

✅ Backend funcional y en desarrollo activo. CRUD completo de roles, usuarios y productos listo.
