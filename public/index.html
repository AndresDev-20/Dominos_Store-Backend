📡 Backend API - Dominós Store 👨‍💻 En Desarrollo 👨‍💻

Bienvenido al backend del ecommerce "Dominós Store". Aquí encuentras una descripción rápida del servidor y sus rutas disponibles.

📦 Descripción

API REST construida con Node.js, Express y Sequelize (ORM con CLI y migraciones), conectada a MySQL. Permite manejar usuarios, roles, productos, carritos, direcciones, compras y más.

🛠 Tecnologías utilizadas

  • Node.js
  • Express.js
  • MySQL
  • Sequelize CLI
  • Bcrypt
  • JWT (Json Web Token)
  • Nodemon

⚙️ Configuración rápida

  • git clone https://github.com/tu-usuario/dominos-store-backend.git
  • npm install
  • Configurar config/config.js con tu conexión a MySQL
  • npx sequelize db:create
  • npx sequelize db:migrate
  • npm run dev

🚀 Rutas disponibles (v1)

🔐 Roles

  • GET /api/v1/roles
  • POST /api/v1/roles
  • PUT /api/v1/roles/:id
  • DELETE /api/v1/roles/:id

👤 Usuarios

  • GET /api/v1/users
  • GET /api/v1/users/:id
  • POST /api/v1/users (Crea usuario + carrito si es cliente)
  • POST /api/v1/users/login
  • PUT /api/v1/users/:id
  • DELETE /api/v1/users/:id

📦 Productos

  • GET /api/v1/products
  • GET /api/v1/products/:id
  • POST /api/v1/products
  • PUT /api/v1/products/:id
  • DELETE /api/v1/products/:id

🛒 Carrito

  • GET /api/v1/carts
  • GET /api/v1/carts/:id
  • POST /api/v1/carts
  • PUT /api/v1/carts
  • DELETE /api/v1/carts

🧾 Detalles del Carrito

  • GET /api/v1/detallecarts
  • GET /api/v1/detallecarts/:id
  • POST /api/v1/detallecarts
  • PUT /api/v1/detallecarts
  • DELETE /api/v1/detallecarts
